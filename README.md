# copy-file
## AIM:
To write a python program for copying the contents from one file to another file.
## EQUIPEMENT'S REQUIRED: 
PC
Anaconda - Python 3.7
## ALGORITHM: 
### Step 1:
open a text file as a read mode

### Step 2: 
open another text file as a write mode
 
### Step 3: 
read a text file

### Step 4:  
and write it in a text file

### Step 5: 
end the program
